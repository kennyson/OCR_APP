Box PDF OCR

Extract text from any PDF stored in Box — powered by Box AI

Server is running Follow the steps below to connect this server to your Box account as a Web App Integration.

How it works

Once configured, right-clicking any PDF in Box.com will show an "OCR Document" option. Clicking it opens this app, which uses Box AI to extract all text from the PDF and displays it in your browser. No files are stored — everything happens in real time.

Setup: Register as a Box Web App Integration

  1. Create a Box Custom App

    Go to the Box Developer Console, click Create New App, choose Custom App, then select User Authentication (OAuth 2.0).

  2. Set required scopes

    Under Configuration → Scopes, enable:
    • Read all files and folders stored in Box
    • Use AI features

  3. Add a Web App Integration

    Go to Configuration → Web App Integrations and click Create a Web App Integration.

  4. Configure the integration

    Fill in the fields:

    • App Name: OCR Document
    • Description: Extract text from this PDF using Box AI
    • File extensions: pdf
    • Execute URL:
    http://YOUR-DOMAIN/ocr?file_id={file_id}&access_token={access_token}&file_name={file_name}

    Replace YOUR-DOMAIN with this server's publicly accessible hostname and port (e.g. from ngrok for local testing, or your deployed URL).

  5. Save and publish

    Save the integration. It will appear in Box under the Open With or right-click context menu for PDF files.

Local testing tip: Run npx ngrok http 3000 to expose this server publicly, then use the generated HTTPS URL in the Execute URL above.

Requirements

  • Box account with AI features enabled (Business Plus or Enterprise plan)
  • Node.js 18+
  • This server accessible via a public HTTPS URL
